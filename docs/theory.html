Poincare map and limit cycle

We consider a Poincare section HHH defined by

H={Φ:φ(Φ)=φ(Φ0)mod  2π}.H =\{ \Phi : \varphi(\Phi)= \varphi(\Phi_0) \mod 2\pi \}.H={Φ:φ(Φ)=φ(Φ0​)mod2π}.

  • φ\varphiφ denotes the global phase.
  • Φ0\Phi_0Φ0​ - phase vector at initial time t0t_0t0​.
  • HHH is a (N−1)(N-1)(N−1)-dimensional hypersurface in NNN-dimensional phase space.
  • In case of φ\varphiφ - mean phase, HHH is a hyperplane.

For each of m-twists solutions, we anticipate a corresponding limit cycle CkC_\mathbf{k}Ck​, piercing HHH close to Φk\Phi_kΦk​ i.e.

Ck∩H0=Φk+E∗,C_\mathbf{k} \cap H_0 = \Phi_\mathbf{k} + \mathbf{E}^*,Ck​∩H0​=Φk​+E∗,

with a small correction vector E∗\mathbf{E}^*E∗. The reason for the small correction E∗\mathbf{E}^*E∗ is that the calibration of active driving forces causes small phase-dependent variations of the instantaneous phase speed (a cilium will speed up at one part of the beat cycle, and slow down at another; but then the phase difference will vary during the cycle, therefore we don't get a perfect m-twist, which has constant phase difference).

Therefore, the first step is to find Φk∗=Φk+E∗\Phi^*_\mathbf{k}=\Phi_\mathbf{k} + \mathbf{E}^*Φk∗​=Φk​+E∗.

We define Poincare map L:H→H\mathcal{L}: H \rightarrow HL:H→H as L(Φ0)=Φ1,\mathcal{L}(\Phi_0) = \Phi_1,L(Φ0​)=Φ1​, where

  • Φ0=Φ(t0)\Phi_0 = \Phi(t_0)Φ0​=Φ(t0​) - some initial phase. Note that for any Φ0\Phi_0Φ0​ we can define a Poincare section as defined above.
  • Φ1=Φ(t1)\Phi_1=\Phi(t_1)Φ1​=Φ(t1​) - where t1t_1t1​ is the next time when our phase trajectory hits the Poincare section.

Then Φk∗\Phi^*_\mathbf{k}Φk∗​ is a fixpoint of Poincare map ( TODO: explain why it cannot be another limit cycle in Poincare plane. )

Procedure to find the fixpoint Φ∗\Phi^*Φ∗

By definition, the fixpoint is such a point that L(Φ∗)=Φ∗\mathcal{L}(\Phi^*) = \Phi^*L(Φ∗)=Φ∗ We define (TODO: this function can be useful in other places - keep this notation?) D(Φ)=L(Φ)−ΦD(\Phi) = \mathcal{L}(\Phi) - \PhiD(Φ)=L(Φ)−Φ and d(Φ)=∥D(Φ)∥2d(\Phi) = \lVert D(\Phi) \rVert ^ 2d(Φ)=∥D(Φ)∥2 Function d(Φ)d(\Phi)d(Φ) is the squared distance between a phase vector and its Poincare map image.

Properties of norm imply that

  • d(Φ)≥0d(\Phi) \geq 0d(Φ)≥0
  • d(Φ)=0  ⟺  Φ=Φ∗d(\Phi)=0 \iff \Phi = \Phi^*d(Φ)=0⟺Φ=Φ∗ - a fixed point

Linearized Poincare map

For any point Φ0\Phi_0Φ0​ which belongs to the Poincare section, we can approximate the Poincare map in vicinity of this point using linear expansion (first order multivariate Taylor expansion):

L(Φ0+Δ)=L(Φ0)+DL(Φ0)+O(∣Δ∣2),\mathcal{L}(\Phi_0+ \Delta) = \mathcal{L}(\Phi_0) + \mathrm{D}\mathcal{L}(\mathbf{\Phi_0}) + \mathcal{O}(|\Delta|^2),L(Φ0​+Δ)=L(Φ0​)+DL(Φ0​)+O(∣Δ∣2), , where Δ\DeltaΔ is a small perturbation.

Let's use this in vicinity of a fixpoint Φ∗\Phi^*Φ∗, taking into account that L(Φ∗)=Φ∗\mathcal{L}(\Phi^*) = \Phi^*L(Φ∗)=Φ∗

L(Φ∗+Δ)=Φ∗+DL(Φ∗)+O(∣Δ∣2).\mathcal{L}(\Phi^* + \Delta) = \Phi^* + \mathrm{D}\mathcal{L}(\mathbf{\Phi^*}) + \mathcal{O}(|\Delta|^2).L(Φ∗+Δ)=Φ∗+DL(Φ∗)+O(∣Δ∣2).

Let's further denote L=DL(Φ∗)\mathbf{L} = \mathrm{D}\mathcal{L}(\mathbf{\Phi^*})L=DL(Φ∗) - linearized Poincare map at a fixed point (matrix). In code Lmat
